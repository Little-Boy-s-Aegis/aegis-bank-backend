# Aegis Bank Backend

This is the Core Banking REST API backend for the Aegis Secure Banking Platform. Built using Spring Boot, Hibernate/JPA, and H2 database.

## Key Features
- JWT-based authentication
- Security Threat Simulation & Defensive Shields (SQL Injection, XSS, BOLA/IDOR, Parameter Tampering)
- Real-time security attack logging
- Multi-currency account balance querying and transaction management
